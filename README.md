# chirps
Chirps - Totally not a twitter clone

Installation steps

1. Install or update RVM
  https://rvm.io/rvm/install
  1.1 Upgrade RVM if already installed (https://rvm.io/rvm/upgrading)
    `> rvm get stable`

2. Install Ruby 3.0.1
  `> rvm install '3.0.1'`

3. Install Rails 6.1.3.1
  `> gem install rails`

4. Start the server
  `> rails s`
